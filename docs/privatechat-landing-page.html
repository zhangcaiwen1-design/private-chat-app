安心聊天隐私政策

更新日期：2026年6月13日

本政策适用于“安心聊天”App。运营主体为南京文晶科技有限公司。联系邮箱：zhangcaiwen1@126.com。

1. 我们收集的信息

  • 账号信息：手机号、昵称、头像地址，用于注册、登录和识别账号。
  • 设备信息：本机生成的设备标识，用于登录状态、账号安全和多设备识别。
  • 聊天内容：你发送或接收的文字、图片、语音、表情、联系人关系和消息时间，用于提供聊天、云端保存与恢复功能。
  • 会员状态：会员套餐、有效期、订单状态，用于判断服务权益。
  • 日志信息：接口请求状态、错误信息等基础运行日志，用于排查故障和保障服务稳定。

2. 权限使用说明

  • 相机：用于扫描好友二维码，以及在聊天中拍摄并发送图片。
  • 相册：用于从相册选择图片并发送给聊天联系人。
  • 麦克风：用于录制并发送语音消息。
  • 剪贴板：用于复制用户 ID，不会主动读取无关剪贴板内容。

3. 信息如何使用

我们使用上述信息来提供账号登录、联系人管理、消息收发、云端记录、设备安全、客服支持、故障排查和服务改进。我们不会将你的聊天内容用于广告画像。

4. 信息存储和保护

你的部分数据会保存在本机，部分数据会保存在服务器，用于跨设备同步和云端恢复。我们会采取访问控制、接口鉴权、数据隔离等措施降低未授权访问风险。请理解，互联网服务无法保证绝对安全。

5. 第三方服务

当前 App 主要使用 Expo/React Native 相关基础能力、Apple 系统能力和自有后端服务。如后续接入统计、支付、客服或崩溃分析 SDK，我们会更新本政策并在必要时取得你的同意。

你可以通过 App 内功能删除消息、联系人和云端记录。若需要注销账号或删除账号相关数据，请发送邮件至 zhangcaiwen1@126.com，邮件标题建议为“账号注销/数据删除申请”，并提供注册手机号用于核验。
