# Apple Developer 注册指南

更新时间：2026-06-13

## 官方入口

- Apple Developer Program 注册入口：https://developer.apple.com/programs/enroll/
- App Store Connect 登录入口：https://appstoreconnect.apple.com/
- Apple Developer 官网：https://developer.apple.com/

## 你要选哪种账号

如果只是个人先上架，可以选 Individual。  
如果要用公司主体、后续做长期商业化、协议和开发者名称统一，建议选 Organization。

## Individual 个人账号需要

1. Apple ID。
2. 开启双重认证。
3. 真实姓名、地址、电话。
4. 支付 Apple Developer Program 年费。
5. 用 Apple Developer App 或网页完成注册。

缺点：App Store 开发者名称会显示个人姓名。

## Organization 公司账号需要

1. 公司 Apple ID，建议不要用个人生活 Apple ID。
2. 公司法人主体信息。
3. D-U-N-S Number（邓白氏编码）。Apple 会用它验证公司身份。
4. 公司官网或可验证联系方式。
5. 你需要有代表公司签署协议的权限。
6. 支付 Apple Developer Program 年费。

优点：App Store 开发者名称显示公司名称，更适合长期经营。

## 推荐流程

1. 先确认是否已有公司主体，以及公司名称是否准备长期使用。
2. 如果公司主体还没确定，先用个人账号可以更快测试 TestFlight，但正式商业化建议最终转公司主体。
3. 打开 https://developer.apple.com/programs/enroll/。
4. 登录 Apple ID。
5. 选择 Individual 或 Organization。
6. 按页面要求填写资料并支付。
7. 注册通过后，登录 https://appstoreconnect.apple.com/。
8. 在 App Store Connect 创建 App，Bundle ID 使用 `com.privatechatapp.calculator`。

## 当前项目需要你确认

- 你打算用个人主体还是公司主体注册？
- 如果用公司主体，公司全称是什么？
- 客服邮箱准备用哪个？
- 隐私政策和用户协议准备挂到哪个域名？

## 注意

Apple 注册、付款、协议签署属于外部账号和付费操作，必须由你本人完成。我可以继续帮你准备资料、配置项目、生成构建包和提交前检查。
