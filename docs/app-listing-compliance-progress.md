## 3. 依赖关系

```text
阶段 1 公司主体调整
  -> 新营业执照

  -> 软著 / ICP备案 / 开发者账号 / 支付商户号

  -> 截图素材
  -> 权限清单

软著 + ICP/App备案 + 开发者账号 + 商店素材
  -> 国内安卓商店提交
```

## 4. 每次查看时的核对顺序

1. 先看“当前一眼看板”。
2. 找到当前阶段。
3. 只推进当前阶段的 🟡 事项。
4. 如果没有 🟡，选择当前阶段第一个 ⬜ 事项。
5. 如果遇到 🔴，先解决阻塞，不要跳到后面。
6. 每完成一项，把状态改成 ✅，并更新下一步。

## 5. 下一次建议动作

下次打开本文件时，直接从这里开始：

1. 确认公司名称备选是否已经准备好。
2. 确认经营范围增项清单是否已经发给代账或提交政务系统。
